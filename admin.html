Admin deployment (Google Workspace)

  1. Admin console → Apps → Google Workspace Marketplace apps.
  2. Add app to domain install list → find “Sections & TOC for Google Slides”.
  3. Review scopes (file-scoped Drive, Slides, script UI) → Allow.
  4. Choose install target (Everyone/all OUs) → Confirm.

If Marketplace installs are blocked, allowlist this app. Ensure Google Slides API isn’t blocked by firewall.
