Setup

  1. Install the add-on from the Google Workspace Marketplace.
  2. Open any Slides deck → Extensions → Add-ons → Sections & TOC → Start.
  3. Choose Tag mode (recommended).
  4. Select slides in the filmstrip → click Bulk-tag selected.
  5. Click Refresh in the sidebar to reload sections.
  6. Click Build / Update TOC to insert a linked TOC slide.

Permissions: Uses file-scoped Drive access (drive.file) and Slides scopes to read/write speaker notes and shapes; email is used only to personalize and (if enabled) check license.
